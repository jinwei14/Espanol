## Espanol：

-  el niño 男孩
-  La niña (the girl)
- La mujer (the woman)
- el hombre (The man)
- Yo soy un niño (I am a boy)
- ¡vamos a ello (go for it)

Non ended with -o masculine
Non ended with -a/-dad/-ción feminine 
Non ended with -e depends 
-s -es plural.

ñ -> sounds like ny. año,mañana->year,tomorrow
